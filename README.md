# J-Blog

## Description

<p>Welcome to <b>J-Blog</b>, a Microblog for user submitted posts!</p>
<p>Front-end built with JavaScript, React and React-Quill. Backend uses Express and JWT for authorization. Database is built with MySQL.</p>
